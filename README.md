Угадай число

Консольная игра на Python. Компьютер загадывает случайное число в выбранном диапазоне, пользователь угадывает его за минимальное количество попыток, получая подсказки "больше" или "меньше". Результаты сохраняются в SQLite и доступны в виде таблицы рекордов.

Требования

Python 3.9 или выше.
pytest, pytest-cov для запуска тестов.

Установка зависимостей

    pip install -r requirements.txt

Запуск игры

    python main.py

Запуск тестов

    pytest

Проверка покрытия

    pytest --cov=. --cov-report=term-missing

Структура проекта

    game.py            игровая логика: генерация числа, проверка догадки, подсказки, ведение раунда
    db.py              работа с SQLite: сохранение результатов, таблица рекордов
    main.py            точка входа, консольное меню
    tests/             модульные тесты (test_game.py, test_db.py)
    requirements.txt   зависимости
    pytest.ini         конфигурация pytest
    README.md          описание проекта
    TEST_CHECKLIST.md  чек-лист приёмочного тестирования
    SUPPORT_LOG.md     журнал обращений поддержки
    REPORT.md          итоговый отчёт

Архитектура

Приложение разделено на три слоя.

Слой данных (db.py) отвечает исключительно за запись и чтение таблицы scores в файле scores.db. Принимает db_path как необязательный параметр, что позволяет тестам использовать временные базы данных.

Слой логики (game.py) реализует правила игры без какой-либо зависимости от ввода-вывода. Функция play_round принимает input_func и print_func как инъецируемые зависимости, что делает её полностью тестируемой без патчинга.

Слой представления (main.py) соединяет два слоя через консольное меню. Не содержит бизнес-логики.

Схема взаимодействия

    Пользователь -> main.py (меню) -> game.py (логика раунда) -> db.py (сохранение) -> scores.db

Пользовательские истории (MVP)

Как игрок, я хочу попытаться угадать число, получая подсказки больше/меньше.
Как игрок, я хочу выбрать уровень сложности (диапазон числа) перед игрой.
Как игрок, я хочу видеть подсказки о близости ("горячо", "тепло") чтобы получать удовольствие.
Как игрок, я хочу видеть таблицу рекордов, отсортированную по минимальному числу попыток.
Как игрок, я хочу проверить свой личный лучший результат по имени.

Диапазоны (уровни сложности)

    1-50    легко
    1-100   стандарт
    1-200   сложно
    1-1000  легенда
